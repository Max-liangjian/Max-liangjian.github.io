Go语言学习

2019-06-01

Go 语言的起源

1、多核硬件架构
2、超大规模分布式计算集群
3、web模式导致的前所未有的开发规模和更新速度

Go 语言创始人

1、Rob Pike  ---  Unix的早期开发者、UTF-8创始人
2、Ken Thompson  ---  Unix的创始人、C语言创始人、
3、Robert Griesemer  ---  Google V8 JS English 和 Hot Spot开发者

Go 语言有点

1、简单：关键字25个；并发编程，内存管理，Go 语言有内置的并发支持及 GC
2、高效：编译的静态类型语言，并且可以通过指针进行直接内存访问
3、生产力：简单清新的依赖管理，简单清新的语法，以及独特的接口类型

四、编写测试程序

1、源码文件名称以_test结尾：xxx_test.go
2、测试方法名以Test开头：func  TestXXX(t  *testing.T){....}
3、引入包：package try_test
4、命令行检查：go test -v
5、编辑器自带保存需设置
6、输出通常用t.Log()

Go 语言编码规则

1、源码放在src下
2、文件扩展名为go
3、为程序建包package

应用程序入口(go文件中的包和方法)

1、必须是main包：package main
2、必须是main方法： func main（）
3、文件名不一定是mian.go

退出返回值（与其他主要编程语言的差异）

1、Go中main函数不支持任何返回值
2、通过os.Exit来返回状态

获取命令行参数（与其他主要编程语言的差异）

1、main函数不支持传入参数  func main(不用参数)
2、在程序中直接通过os.Args获取命令行参数

变量的声明与赋值

1、var a int = 1
2、var（
a int = 1
b int = 1
）
3、a := 1
4、全局声明变量和类型 var a int
a = 1 通常用于全局
5、与其他主要编程语言的差异：赋值可以进行自动判断类型；在一个赋值语句中可以对多个变量进行同时赋值

常量定义

1、与其他主要编程语言的差异：可以快速设置连续值（iota + 1）连续加1  
2、常量的定义和初始化方式
3、自增常量iota默认为0

基本数据类型

1、bool
2、string
3、int  int8  int16 int 32 int 64 (数字表示类型所代表的位数，不带数字即为默认)
4、uint unit8 uint16 uint32 uint64 uintptr
5、byte   //alias for int 8
6、rune  //alias for int32  represents a Unicode code point
7、float32 float64
8、complex64 complex128
9、注意：①个、Go语言不允许隐式类型转换（eg：int  和  int32之间不能转换）  ②别名和原有类型也不能进行隐式类型的转换（可以做显示的转换 a = int64(b) ）

类型的预定义值（math）

1、math.MaxInt64
2、math.MaxFloat64
3、math.MaxUint32

指针类型

1、不支持指针运算
2、string是值类型，其默认的初始化值为空字符串，而不是null，长度为0
3、输入变量的类型t.Logf("%T %T", a, aPtr)

运算符

1、算术运算符
①Go语言没有前置的++，--（++a❎）
②自增自减（后置的）a := 10; a++;  => 11
2、比较运算符
①有部分特殊点后期谈到
②用==比较数组 ：相同维数且有相同个数的元素的数字才可以比较；每个元素都相同的才相等； 
3、逻辑运算符：没有本质差别
4、位运算符：与、或、异或、同或
①&^ 按位置零（左右两个操作位:右边操作位为1值为0；右边的操作位为0值为左边的操作位 ）

数组

1、定义数组： a := [...]int{1,2,3,4}
2、长度不同的同维数组不能比较
3、声明数组：①var a[3]int ; a[0]=1(先声明并初始化为默认0值，后赋值)；②a := [3]int{1,2,3}(声明同时初始化)；③a := [2][2]int{{1,2},{3,4}}(多维数组初始化)

循环

1、Go语言仅支持循环关键字for
2、for前后的（）不需要了
3、条件循环 :
4、无线循环：

if条件

1、关键字if、else （括号省略不需要）
2、condition条件内容表达式必须为布尔值
3、支持变量赋值
4、if a := 1 == 2;（变量赋值） a(必须为布尔值) {}

switch条件

1、 条件表达式不限制为常量或者整数
2、单个case中，可以出现多个结果选项，使用逗号分隔符 case1,2,3  某值等于1，2，3
3、与c语言等规则相反，Go语言不需要用break来明确退出一个case
4、可以不设定switch之后的条件表达式，在此种情况下，整个switch结构与多个if..else if的逻辑作用等同
